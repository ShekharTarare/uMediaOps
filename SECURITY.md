# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in uMediaOps, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, please email: **security@shekhartarare.dev**

You should receive a response within 48 hours. We will work with you to understand the issue and address it promptly.

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.x     | ✅        |

## Scope

This policy applies to the uMediaOps NuGet package and its source code in this repository.
